Facsimile ${version} Release Notes
=============================

This is an alpha release of the Facsimile simulation/emulation library.  Until
Facsimile has reached a stable release version, there will be no guarantees
made concerning backwards compatibility with earlier releases; all public
interfaces should be regarded as fluid and subject to change.

It is strongly recommended that you do not use this release of Facsimile in a
production environment  This release is provided to solicit feedback and to
encourage interested parties the kick Facsimile's tires and get involved in the
project.

Please report any and all bugs found using the GitHub-hosted issue-tracking
system at:

    http://github.com/Facsimile/facsimile/issues

Your feedback is both welcomed and encouraged.  If you have any questions or
comments about Facsimile, feel free to report them via the Facsimile Google
Group:

    https://groups.google.com/forum/?fromgroups#!forum/facsimile-users
